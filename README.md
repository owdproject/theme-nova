<h3 align="center">
  Nova Theme for your Open Web Desktop client.
</h3>

## Overview

The main theme for Open Web Desktop.

## Installation

```bash
pnpm desktop add @owdproject/theme-nova
```

## Usage

#### Configuration

```typescript
// desktop.config.ts
export default defineDesktopConfig({
  theme: '@owdproject/theme-nova',
  apps: [],
})
```
